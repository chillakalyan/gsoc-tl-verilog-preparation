## Simulation

The TL-Verilog implementation of this example can be simulated in Makerchip.

**Makerchip Simulation Link**

```
https://makerchip.com/sandbox/031fmhvgg/0g5hAxE
```

This simulation demonstrates a simple 4-stage pipeline:

Fetch → Decode → Execute → Writeback

The waveform shows how the program counter (`pc`), instruction (`instr`), and result signals move through the pipeline stages across clock cycles.
