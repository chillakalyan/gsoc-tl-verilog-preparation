# Makerchip Learning Notes

## Introduction

Makerchip is an online IDE used for designing, simulating, and visualizing TL-Verilog and Verilog designs.

It allows developers to quickly test hardware designs and observe waveform outputs and pipeline visualizations.

Website:
https://makerchip.com

---

## Features

### 1. Interactive Simulation

Makerchip allows users to simulate hardware designs directly in the browser without installing any local tools.

Features include:

- waveform visualization
- signal tracing
- pipeline stage visualization

---

### 2. Sandbox Environment

The sandbox environment allows developers to quickly prototype designs.

Example:
https://makerchip.com/sandbox

Users can paste TL-Verilog code and run simulations immediately.

---

### 3. Visualization

Makerchip integrates visualization tools that show how signals propagate through pipeline stages.

This helps developers understand complex pipeline behavior.

## Workflow

Typical workflow for using Makerchip:

1. Write TL-Verilog code
2. Paste code into Makerchip sandbox
3. Run simulation
4. Analyze pipeline visualization
5. Refactor design

---

## Benefits for Hardware Design

Makerchip provides several advantages:

- no local setup required
- quick experimentation
- visual debugging
- easy sharing of simulation links
