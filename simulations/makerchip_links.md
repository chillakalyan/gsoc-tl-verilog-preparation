## How to Run the Simulations

1. Click any Makerchip link above.
2. The design will open in the Makerchip editor.
3. Click **Run** to execute the simulation.
4. Observe the waveform or visualization depending on the example.

---

## Notes

* Some examples include **visualizations** (Boat, Solar System, Sunrise Sunset).
* Pipeline examples demonstrate how **TL-Verilog simplifies pipeline design**.
* RISC-V examples show how TL-Verilog can be used for **processor architecture design**.
